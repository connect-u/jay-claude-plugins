이 저장소의 워크트리는 .claude/worktrees/ 아래(메인 루트 하위)에 생성된다. engram의 프로젝트 판별이 "최근접 .memory 마커 상향 탐색 → git root" 순서라서, 워크트리 안에서 시작한 세션도 상향 탐색이 메인 루트의 .memory에 먼저 닿는다 — 결과적으로 모든 워크트리가 하나의 프로젝트 메모리를 공유한다.

git root 기준이었다면 워크트리마다 메모리가 갈라졌을 것. 단, 메인 루트에 .memory가 없는 상태에서 워크트리 세션이 첫 project 쓰기를 하면 git root(=워크트리 디렉토리)에 .memory가 생겨 갈라진다 — 프로젝트에서 engram을 켤 때는 메인 루트에 mkdir .memory를 먼저 하는 게 안전하다.
